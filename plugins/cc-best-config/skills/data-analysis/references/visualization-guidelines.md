# Visualization Guidelines

Choose visuals that clarify the point. Do not add charts just to make the report look busy.

## Prefer

- line chart: change over time
- bar chart: compare categories
- stacked bar: show composition across categories when segment count is small
- histogram: show distribution
- box plot: compare distributions and outliers
- table: exact values, source comparison, or compact synthesis

## Avoid

- pie charts for complex comparisons
- 3D charts
- overloaded dashboards inside a report
- multiple charts that repeat the same message

## Good chart behavior

- Write a title that states the takeaway.
- Label units clearly.
- Keep colors simple and purposeful.
- Annotate only what matters.
- Put a short interpretation under the figure when needed.

## Research-material reports

For non-tabular analysis, prefer:

- source comparison tables
- claim-evidence matrices
- timeline tables
- category summary tables

Use figures only when they reveal structure better than prose or tables.
